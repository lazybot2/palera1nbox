* Turdus Function Menu

> Function menu (Option): Safe Mode - Normal Start - In Sileo (install Sileo in jailbroken environment, please do not select Cydia if there is one) - SHSH (perfect downgrade with SH certificate) - Revert (clear boot save of connected phone on the box) Choose nothing to automatically boot into the jailbreak environment. > The box supports commonly used 2.4G wireless network cards by default. If a valid wireless network card is found during startup and there is no network connection, an attempt will be made to connect to a WiFi with SSID: lazybot and password: 12345678. > Supports hot swappable USB drives or card readers, and supports exfat, ntfs, ext3, ext4 file formats.

> If you need to back up the boot archive: please create a lazybot_back folder in the root directory of the USB drive and run turdus. The box will be automatically backed up. To upload the boot file to the box, you need to store the file in the lazybot.load folder in the root directory of the USB drive. And run Turdus.

> Upload IPSW flashing package: Store the IPSW files to be uploaded on the USB drive. And name it in the following format: "6s. ipsw" "6sp. ipsw" "5se. ipsw" "ipad97. ipsw" "ipad5. ipsw" "i7. ipsw" "i7p. ipsw" "ipod7. ipsw" "ipad7. ipsw" "ipadp129-2.ipsw" "ipadp10. ipsw" If you do not know the required IPSW package naming format. > Wait for the box to downgrade or boot up. If it is the first time to downgrade, you need to connect to the internet to flash the device. After activation is complete, enter Turdus to run jailbreak boot up. After the first jailbreak boot up, you need to run checkra1n mode again after the phone enters the desktop normally, and load the black flag icon onto the phone. )

* SHSH downgrade

> Need to prepare a USB flash drive, insert it and run turdus mode to select the SHSH option

> Connect your phone and enter DFU mode.

> After the box runs, a folder with the phone ECID number will be created on the USB drive. Exit Turdus. Pull out the USB drive. (It can also be created on a mobile phone)

> Put the corresponding certificate file ***. shh2 and the corresponding flashing package ***. ipsw in the corresponding folder

> Insert the USB drive and connect to the network. Connect your phone and enter DFU mode. Run Turdus mode and select the SHSH option. Follow the box prompts multiple times to enter DFU mode. Until the phone is turned on normally.
